# 🚗🧠 NeuroEye Controlled Toy Car

## 📌 Overview

The Neuro-Eye Controlled Toy Car initiative leverages Brain-Computer Interface (BCI) and Electrooculography (EOG) technologies to create an innovative dual-input assistive control system. This system is designed to help individuals with mobility impairments by enabling control through brainwaves and eye movements. EEG signals are used to execute forward and stop commands, while EOG signals detect left and right eye movements (horizontal) to navigate directionally. The integration of these non-invasive technologies provides an intuitive and accessible interface for users with limited physical capabilities.


## 🎯 Objective

To develop a system that enables **hands-free control of a vehicle using brain activity and eye signals**.

## 🧩 Hardware Components

1. Arduino Uno Microcontroller
2. BioAmp EXG Pill (2)
3. BioAmp Cables
4. Electrodes
5. Jumper Cables
6. Conductive Gel
7. Computer System
8. RF Module
9. Motor Driver
10. HT12D (Decoder), HT12E (Encoder)
11. Toy Car


## ⚙️ System Architecture

1️⃣ EEG signals are captured using electrodes placed on the user’s head.
2️⃣ The signals are amplified using BioAmp EXG Pill modules.
3️⃣ The microcontroller processes the signals.
4️⃣ RF modules transmit commands to the toy car.
5️⃣ The motor driver controls the movement of the car.


## 🚦 Working

* ▶ **Forward** – The car moves forward when the signal is detected.
* ⬅ **Left** – The car turns left.
* ➡ **Right** – The car turns right.
* ⏹ **Stop** – The car stops.

## 🌍 Applications
